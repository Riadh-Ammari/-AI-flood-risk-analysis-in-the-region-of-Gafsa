# Outputs Directory

This directory contains all results from the flood risk analysis pipeline.

## Directory Structure

### `/maps/` - GeoTIFF Risk Maps
- `flood_susceptibility_gb_class.tif` - Risk classification (0=Low, 1=Medium, 2=High, 3=Very High)
- `flood_susceptibility_gb_prob.tif` - Risk probability map (0-1 scale)
- `flood_susceptibility_rf_class.tif` - Random Forest predictions

**Note:** These files are excluded from Git due to large size (28+ MB). Regenerate by running `04_flood_susceptibility_mapping.py`

### `/visualizations/` - Result Visualizations
- `model_training_results.png` - ML model performance (accuracy, confusion matrix)
- `flood_susceptibility_maps.png` - 4-panel flood risk visualization
- `building_exposure_analysis.png` - Buildings at risk by category
- `delegation_risk_analysis.png` - Choropleth maps by delegation
- `priority_delegations.png` - Top 5 high-risk delegations
- `flood_validation_results.png` - Historical flood validation
- `feature_engineering_results.png` - Feature distributions
- `04_features_summary.png` - 9-panel feature visualization
- `05_correlations.png` - Feature correlation heatmap
- `risk_score_distribution.png` - Risk category distribution

### `/reports/` - Analysis Reports
- `data_statistics_report.md` - Data quality and statistics (99.9% completeness)
- `delegation_risk_report.md` - Per-delegation risk assessment
- `flood_validation_report.md` - Model validation against historical floods
- `feature_engineering_summary.txt` - Feature extraction summary

### `/statistics/` - Numerical Results
- `delegation_risk_statistics.csv` - Risk metrics for 10 delegations
- `flood_susceptibility_statistics.csv` - Area distribution by risk class
- `flood_validation_details.csv` - Historical event validation scores
- `data_statistics_report.json` - Data quality metrics (machine-readable)
